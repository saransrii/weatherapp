# 🌟 Overview

The **Weather App** is a lightweight web application developed using **HTML, CSS, and JavaScript**. It enables users to search for any city and instantly retrieve live weather information such as temperature, weather conditions, humidity, and wind speed.

The application communicates with the **OpenWeatherMap API**, ensuring that users receive accurate and up-to-date weather information in real time.

Designed with simplicity and usability in mind, the project demonstrates API integration, asynchronous JavaScript programming, and responsive web design.

# ⚙️ Application Workflow

The application follows a simple workflow:

1. User enters a city name.
2. The search request is validated.
3. A request is sent to the OpenWeatherMap API.
4. Weather information is received in JSON format.
5. The data is processed and displayed.
6. Smooth animations reveal the weather information.
7. Errors are handled gracefully if the city is invalid.
